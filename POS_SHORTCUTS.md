# POS System Keyboard Shortcuts

This guide outlines the keyboard shortcuts available on the Point of Sale (POS) page for high-speed operation.

## 🚀 Core Navigation
| Key | Action | Description |
|-----|--------|-------------|
| **F1** | Focus Search | Immediately moves the cursor to the product search / barcode input. |
| **F2** | Assign Customer | Opens the customer lookup modal to link a sale to a profile. |
| **Esc** | Close / Cancel | Closes any open modal (Payment, Customer, Receipt) or cancels the current action. |

## 💰 Checkout & Quotes
| Key | Action | Description |
|-----|--------|-------------|
| **F8** | Save Quote | Saves the current cart as a draft quote without processing payment. |
| **F9** | Quick Pay | Opens the Payment Modal and sets the method to "Cash". |
| **F10** | Complete Sale | Processes the final payment if the balance is fully covered in the modal. |

## 🛒 Cart Management
| Key | Action | Description |
|-----|--------|-------------|
| **Ctrl + Alt + C** | Clear Cart | Empties the entire shopping cart (requires confirmation). |

---
*Note: These shortcuts are active globally on the POS page. If a shortcut is not working, ensure the application window is in focus.*
